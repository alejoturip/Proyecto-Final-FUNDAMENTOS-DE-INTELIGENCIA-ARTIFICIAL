# Carpeta del dataset

Aquí van las imágenes de entrenamiento, **una subcarpeta por raza**.
El nombre de la subcarpeta es la etiqueta que aprende el modelo y debe
coincidir exactamente con las claves de `backend/razas.json`.

    imagenes/
      labrador_retriever/   001.jpg  002.jpg  ...  (~150 imágenes)
      golden_retriever/     001.jpg  002.jpg  ...
      pastor_aleman/        001.jpg  002.jpg  ...
      ...

Total esperado: 25 carpetas x ~150 imágenes = ~3.750 imágenes.

Esta carpeta se llena en la **Fase 2** del proyecto.
